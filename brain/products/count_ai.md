# Count AI

## Overview

Count AIは、社内の定型集計・レポーティング業務をAIで自動化するプロダクトです。売上集計、KPIレポート、日次/週次の業務レポート作成など、「毎回同じフォーマットで数字をまとめる」業務を対象にしています。

## Value Proposition

- 手作業でのExcel集計・レポート作成の工数を削減する
- 決まったフォーマットへの数値当てはめではなく、数字の変化点や異常値の指摘まで行う
- 既存の会計・CRM・スプレッドシートデータソースと接続し、レポートを自動生成する

## Use Cases

- 週次営業レポートの自動作成（案件進捗・受注状況の要約）
- 月次売上・KPIレポートの自動生成
- 部門別・顧客別の実績集計
- 異常値・傾向変化のハイライト（例: 特定顧客からの問い合わせ急増）

## 想定顧客像

- 手作業のレポート作成に月あたり数十時間を費やしている企業
- 複数の担当者が個別にExcelで集計しており、フォーマットが統一されていない企業
- 経営会議向けの資料作成に時間がかかっている企業

## Pricing

- PoC型: 月額25万円〜35万円、1〜2ヶ月（対象レポート1〜2種類に絞る）
- 月額伴走型: 月額40万円〜（対象レポート数に応じて変動）

## 導入時の注意事項

- 集計対象データのフォーマットが不統一な場合、PoC期間内にデータ整形の工数が発生する可能性がある
- 経営会議に直結するレポートは、初期は「人によるレビューを前提とした生成」から始め、精度確認後に自動化範囲を広げる
