# テンプレート: 議事録要約

## 用途

商談・打ち合わせの内容を、社内共有用または顧客への確認送付用に要約するためのテンプレート。

## テンプレート本文

```txt
## {{会議名}}（{{日付}}）

### 参加者
- {{参加者一覧}}

### 議論内容
- {{論点1}}
- {{論点2}}

### 決定事項
- {{決定した内容}}

### 次アクション
1. {{担当者}}: {{アクション内容}}（期限: {{期限}}）
2. {{担当者}}: {{アクション内容}}（期限: {{期限}}）

### 保留事項
- {{次回以降に持ち越す論点}}
```

## 記入時のポイント

- 「議論内容」と「決定事項」を分けて書く（議論しただけのものを決定事項に混ぜない）
- 次アクションには必ず担当者と期限を入れる
- 顧客に送付する場合は、社内向けの推測・懸念メモは含めない

## 使用例（Project X 要件確認MTG）

```txt
## Project X 要件確認MTG（2026-07-01）

### 参加者
- 北原氏（Northlight）、Select営業担当、加藤（Select）

### 議論内容
- データ提供範囲について、個人情報を含む部分の扱いを議論
- 費用対効果の説明タイミングについて

### 決定事項
- 個人情報を含まない案件属性情報のみを提供範囲とする方針で合意
- 費用対効果の具体的な試算は次回MTGで提示する

### 次アクション
1. Select（加藤）: 費用対効果の試算を準備（期限: 次回MTG前）
2. Select（営業担当）: 提案書ドラフトを今週中に送付
3. Northlight（北原氏）: データ提供フォーマットの社内確認

### 保留事項
- 問い合わせ対応（フェーズ2）の着手時期
```
