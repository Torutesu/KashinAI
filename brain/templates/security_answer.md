# テンプレート: セキュリティ質問への回答

## 用途

顧客からのセキュリティ・データ取り扱いに関する質問に回答する際の標準テンプレート。company/security_policy.md の内容と整合させて使用する。

## テンプレート本文

```txt
セキュリティに関するご質問をありがとうございます。

現在の弊社の対応方針は以下の通りです。

- データの利用範囲: {{当該案件でのデータ利用範囲を明記}}
- データの保管: {{保管場所・保管期間を明記}}
- データの削除: 契約終了後、{{削除条件・期限}}以内にデータを削除いたします
- アクセス権限: 当該プロジェクトにアサインされたメンバーのみがアクセス可能です
- 外部パートナーの関与: {{関与の有無、関与する場合は事前合意の旨}}

{{個別の懸念点への回答}}

ご不明な点がございましたら、契約書ドラフトの秘密保持条項と合わせてご確認ください。
```

## 記入時のポイント

- 曖昧にごまかさず、具体的に答える
- 未確定の事項は「確認の上回答する」と明示する（断定しない）
- 契約書の秘密保持条項との整合を意識する
- 顧客の業界特性（個人情報保護の厳しさ等）に応じて回答の重点を調整する

## 使用例（Customer A / 人材紹介業向け）

```txt
セキュリティに関するご質問をありがとうございます。

現在の弊社の対応方針は以下の通りです。

- データの利用範囲: 本PoCの対象業務（営業資料作成）にのみ利用いたします
- データの保管: 弊社管理のクラウド環境にて、契約期間中のみ保管いたします
- データの削除: 契約終了後、合意した期限内にデータを削除いたします
- アクセス権限: Project Xにアサインされたエンジニア・営業担当のみがアクセス可能です
- 外部パートナーの関与: 現時点では想定しておりませんが、必要になった場合は事前にご相談の上で進めます

求職者様の個人情報については、今回のPoC範囲から除外し、案件属性情報のみを利用する方針としております。

ご不明な点がございましたら、契約書ドラフトの秘密保持条項と合わせてご確認ください。
```
