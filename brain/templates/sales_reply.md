# テンプレート: 営業返信

## 用途

顧客からの問い合わせ・商談メッセージに対する標準的な返信テンプレート。Slack/メールいずれでも使用可能。

## テンプレート本文

```txt
{{顧客名}}様

ご連絡いただきありがとうございます。

{{顧客からの質問・要望の要約}}について承知いたしました。

{{回答・対応方針を簡潔に}}

つきましては、{{次に確認したいこと・次のステップ}}についてご確認いただけますでしょうか。

引き続きよろしくお願いいたします。
```

## 記入時のポイント

- 冒頭は感謝から始める
- 顧客の質問・要望を一度要約し、正しく理解していることを示す
- 回答は結論から先に述べる
- 曖昧な回答をする場合は「確認の上ご連絡します」と明示する
- 次のアクションを必ず1つ以上含める

## 使用例（Customer A向け）

```txt
北原様

ご連絡いただきありがとうございます。

PoCの費用対効果について、具体的な工数削減見込みをお伝えしたいというご要望、承知いたしました。

現在、営業資料作成にかかる工数をもとに削減見込み時間を試算しております。次回MTGにて具体的な数値とともにご説明させてください。

つきましては、次回MTGの日程調整と合わせて、案件属性データの提供フォーマットについてもご相談させていただけますでしょうか。

引き続きよろしくお願いいたします。
```
