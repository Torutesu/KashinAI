# テンプレート: 提案フォローアップ

## 用途

提案送付後、一定期間返答がない場合、または提案書ドラフトの送付・進捗確認を行う場合のフォローアップテンプレート。催促にならないよう、進捗確認や追加情報提供という切り口を使う。

## テンプレート本文

```txt
{{顧客名}}様

いつもお世話になっております。

先日ご提案させていただいた{{提案内容の名称}}について、{{現在の検討状況を尋ねる一文}}。

こちらでは{{追加で用意した情報・資料があればその説明}}を準備しておりますので、ご都合の良いタイミングで共有させていただければと思います。

もし現時点でご不明点や懸念点がございましたら、遠慮なくお知らせください。

引き続きよろしくお願いいたします。
```

## 記入時のポイント

- 「まだですか」という催促のトーンにしない
- 進捗確認は「追加情報の提供」とセットにして、こちらから価値を提供する姿勢を示す
- 懸念点があれば早めに引き出す一文を入れる

## 使用例（Customer A / Project X向け、提案書ドラフト送付）

```txt
北原様

いつもお世話になっております。

先日ご相談させていただいたAI業務効率化PoCについて、提案書ドラフトを準備いたしましたので本日中にお送りいたします。

費用対効果の試算については次回MTGにて詳細をご説明させていただければと思いますが、ドラフトの時点でもご不明点がございましたら遠慮なくお知らせください。

引き続きよろしくお願いいたします。
```
