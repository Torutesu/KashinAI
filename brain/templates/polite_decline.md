# テンプレート: 丁寧な見送り・お断り

## 用途

提案が予算・タイミング等の理由で見送りとなった場合に、関係性を維持しながら丁寧に対応するためのテンプレート。将来の再提案余地を残すことを目的とする。

## テンプレート本文

```txt
{{顧客名}}様

ご検討いただきありがとうございました。また、率直にご状況を共有いただき感謝いたします。

{{見送り理由への理解を示す一文}}、承知いたしました。

今回はご縁がありませんでしたが、{{再検討のタイミングや条件}}の際には、ぜひまたお話しさせていただけますと幸いです。

{{もし今回のご要望に近い、より小さい/低価格の代替案があれば一言添える}}

引き続きどうぞよろしくお願いいたします。
```

## 記入時のポイント

- 断られたことへの不満や交渉がましさを一切出さない
- 見送り理由（予算・タイミング等）への理解を示す
- 「また次の機会に」で終わらせず、具体的な再検討タイミングの目安を提示する
- 可能であれば、より小さい代替案を軽く提示しておく（押し売りにならない範囲で）

## 使用例（Customer C向け）

```txt
Warmspace 様

ご検討いただきありがとうございました。また、率直にご予算感を共有いただき感謝いたします。

現状のご予算感とのギャップがある中で、今回は見送りとのご判断、承知いたしました。

今回はご縁がありませんでしたが、事業の成長に合わせて予算感が変わるタイミングがあれば、ぜひまたお話しさせていただけますと幸いです。今回のPoC範囲より小さいライトなプランもご用意できる可能性がありますので、その際は改めてご提案させてください。

引き続きどうぞよろしくお願いいたします。
```
