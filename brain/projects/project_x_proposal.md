# Project X — 提案内容

## 提案タイトル

AI業務効率化PoC（株式会社Northlight様向け）

## 課題整理

- 営業資料作成に1案件あたり2〜3時間の工数がかかっている
- 問い合わせ対応の一次回答スピードが競合と比較して遅れている
- 上記により、営業担当が本来注力すべき顧客折衝の時間が圧迫されている

## PoC対象範囲

### フェーズ1（PoC本体）: 営業資料作成の自動化

- 案件情報（業種・職種カテゴリ・求人条件等、個人情報を除く）をもとに、営業資料のドラフトを自動生成
- 想定効果: 1案件あたりの資料作成時間を2〜3時間から30分程度に短縮（想定値、PoCで検証）

### フェーズ2（次PoC候補）: 問い合わせ対応の効率化

- 求職者・企業からの問い合わせに対する一次回答ドラフトの自動生成
- 今回のPoC範囲には含めず、フェーズ1の結果を見て提案する

## 進め方とスケジュール

1. 要件確認（〜2026-07中旬）
2. PoC環境構築・初期実装（2026-08）
3. 試験運用・フィードバック反映（2026-09）
4. 効果測定・PoC結果報告（2026-10）
5. 本契約検討（2026-11以降）

## 体制

- Select側: エンジニア1名（加藤 遼）+ 営業担当1名がPoC期間中伴走
- Northlight側: 北原氏を窓口に、実務担当者2〜3名からのフィードバックを想定

## 価格

- PoC価格: 月30万円（3ヶ月、合計90万円）
- 本契約への移行時は、対象業務範囲の拡張（フェーズ2含む）に応じて別途見積

## 前提条件・確認事項

- 案件属性情報（個人情報を除く）の提供が可能であること
- 週次での短時間フィードバックMTGに対応いただけること
- セキュリティ・データ取り扱いについては別途説明の上、合意すること
